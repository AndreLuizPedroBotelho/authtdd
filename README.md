# Project example using tdd with jest

# Instructions                            

## 1. Install the dependencies

``` 
$ yarn
```

## 2. If not exist , create file **.env** based in file ** .env-example**

``` 

$ cp .env-example .env 

```

## 3. Edit the .env file with the correct information

## 4. Execute the test

``` 

$ yarn test 

```
